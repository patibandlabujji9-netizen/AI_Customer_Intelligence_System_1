# Intelligent Sales Forecasting & Inventory Optimization System

## Project Overview

The Intelligent Sales Forecasting & Inventory Optimization System is an AI-powered business intelligence platform designed to help organizations accurately predict future sales demand, optimize inventory management, and improve supply chain efficiency. The system leverages Machine Learning, Data Analytics, Forecasting Models, and Interactive Dashboards to provide actionable business insights that support strategic decision-making.

Traditional forecasting methods often struggle to adapt to dynamic market conditions, seasonal trends, and changing customer behavior. This solution addresses those challenges by utilizing advanced predictive analytics to forecast demand, reduce stock shortages, minimize overstocking, and maximize profitability.

---

## Problem Statement

Inaccurate sales forecasts lead to significant inventory management challenges, including:

* Stockouts resulting in lost sales opportunities
* Overstocking leading to increased storage and carrying costs
* Poor demand planning and resource allocation
* Reduced operational efficiency
* Lower customer satisfaction

The proposed system uses Artificial Intelligence and Business Intelligence techniques to provide accurate sales predictions and inventory recommendations for improved business performance.

---

## Objectives

* Predict future sales using Machine Learning algorithms
* Optimize inventory levels based on forecasted demand

## Installation Guide

### Clone Repository

```bash
git clone <repository-url>
cd INTELLIGENT_SALES_2
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows:

```bash
venv\Scripts\activate
```

Linux/Mac:

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
